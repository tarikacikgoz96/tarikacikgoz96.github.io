# SAP ABAP Development Workbench Tcode'ları SAP ABAP yazılımcılarının bilmesi gereken Tcode'lardır. SAP ABAP dünyasında çok fazla tcode mevcuttur. Geliştirme yaparken unutmamak, tekrar hatırlamak adına aktif kullanılan tcodeların genel listesidir. Liste zamanla güncellenecektir. ## Development Tcodes - **SE80 :** Paket yönetim ekranı (Tüm dev içeriği) - **SE38 :** Program - **SE24 :** Class - **SE37 :** Function - **SE11 :** Table, Data Element, Domain, Structure, View (DB elemanları) - **SE16, SE16N :** Tablo içeriği verileri - **SE09 :** Sistemdeki Request listesi / Req. oluşturma - **STMS :** Sistem Request Kuyruğu (DEV, TEST, CANLI) - **SM37 :** Job Görüntüleme, inceleme - **SM36 :** Job Oluşturma - **SM12 :** Kilit Kayıtları - Edite Kitlenen programları/nesneleri kullanıcı bazında açma - **SE03 :** Transport Organizer - Hangi nesne hangi transportta/requestte - **SE93 :** Tcode Ekranı - **SE43 :** Area Menü ekranı (Pakete özel menü hazırlama) - **SMARTFORMS :** Smartform oluşturma, değiştirme, oluşturma - **SMARTSTYLES :** Font, Yazı tipi vb. ayarlama - **SE78 :** Logo yükleme ekranı - **SOST :** Gönderilen maillerin kontrolü - **STRUST :** API Bağlantılar için sertifika yükleme - **/IWFND/GW_CLIENT :** OData, RestAPI Test, Oluşturma Ekranı - **SEGW :** OData Service Listesi, Ekranı - **SM59 :** HTTP HTTPS API RFC vb. Bağlantı kontrol ekranı - **SM50 :** Aktif arkada çalışan programların listesi (O anki sistem/sunucu) - **SE63 :** Program Metin Çevirileri (Translate) - **SE51 :** Screen Painter ekranı - **SM30 :** Table Maintence Ekranı - **/n :** Üst ekrana geçiş, geri - **/h :** Anında Debug Birçok tcode/ekrana birçok yerden kod yazmadan da ulaşılabilmektedir.
