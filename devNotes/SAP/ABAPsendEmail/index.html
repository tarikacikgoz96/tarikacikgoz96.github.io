ABAP Send Email

Published: 29.06.2026 Updated: 29.06.2026

Bir ABAP programı içerisinden email yollamanızı sağlar.

Email içeriği HTML olarak yazılır. İçerik HTML olduğu için içeriği görselleştirme özgürlüğü sunar.

Programdan gönderilen emaillerin gönderim durumu/kuyruğu SOST tcode ile kontrol edilir.

Dikkat: Emailin karşı tarafa başarılı şekilde ulaşması için sisteminizin email gönderim ayarlarının düzgün yapılmış olması gerekmektedir aksi halde SOST den başarısız sonucunu alırsınız.

Örnek gönderim görselleri aşağıdaki gibidir.

Temel çalıştırma aşağıdaki gibidir;

*&---------------------------------------------------------------------*
*& Report ZTEMPLATE_EMAIL
*&---------------------------------------------------------------------*
*&
*&---------------------------------------------------------------------*
REPORT ztemplate_email.


START-OF-SELECTION.

  DATA: lt_recievers TYPE soli_tab,
        lv_subject   TYPE char50,
        lt_content   TYPE soli_tab,
        lv_message   TYPE string.

  DATA: lo_cls TYPE REF TO zcl_template.

  CREATE OBJECT lo_cls.



  lt_recievers = VALUE #(
    BASE lt_recievers
    ( line = 'mailaddr1@company.com' )
    ( line = 'mailaddr2@company.com' )
    ( line = 'mailaddr3@company.com' )
  ).

  lv_subject = 'Mail Başlığım: Bu bir Test mailidir.'.

  lt_content = VALUE #(
    BASE lt_content
    "( line = '<style>p { padding: 5px; text-align: left; }</style>' )
    ( line = 'Merhaba,<br/>' )
    ( line = '<br/>' )
    ( line = 'SAP ABAP ile test maili gönderilmiştir.<br/>' )
    ( line = 'Sizde bu metodu kullanarak gönderebilirsiniz.<br/>' )
    ( line = '<br/>' )
    ( line = 'İyi çalışmalar.' )
  ).



  lo_cls->send_email(
    EXPORTING
      it_recievers = lt_recievers  " Mail alıcıları
      iv_subject   = lv_subject    " Mail başlığı
      it_content   = lt_content    " Mail içeriği HTML
    IMPORTING
      ev_message   = lv_message
  ).

  BREAK-POINT.


END-OF-SELECTION.

SEND_EMAIL metodu;

Method Signature:
*IT_RECIEVERS   Importing   Type    SOLI_TAB
*IV_SUBJECT     Importing   Type    CHAR50
*IT_CONTENT     Importing   Type    SOLI_TAB
*EV_MESSAGE     Exporting   Type    STRING

  METHOD send_email.


    DATA: ls_mailsubject    TYPE sodocchgi1,
          ls_mailreciever   TYPE somlrec90,
          lt_mailrecievers  TYPE TABLE OF somlrec90.


    LOOP AT it_recievers ASSIGNING FIELD-SYMBOL(<fs_reciever>).
      CLEAR ls_mailreciever.
      ls_mailreciever-receiver  =  <fs_reciever>.
      ls_mailreciever-rec_type  = 'U'.
      APPEND ls_mailreciever TO lt_mailrecievers.
    ENDLOOP.


    "ls_mailsubject-obj_name  = 'TEST MAIL'.
    ls_mailsubject-obj_langu = sy-langu.
    ls_mailsubject-obj_descr = iv_subject.



    CALL FUNCTION 'SO_NEW_DOCUMENT_SEND_API1'
      EXPORTING
        document_data              = ls_mailsubject
        document_type              = 'HTM'
      TABLES
        object_content             = it_content
        receivers                  = lt_mailrecievers
      EXCEPTIONS
        too_many_receivers         = 1
        document_not_sent          = 2
        document_type_not_exist    = 3
        operation_no_authorization = 4
        parameter_error            = 5
        x_error                    = 6
        enqueue_error              = 7
        OTHERS                     = 8.
    IF sy-subrc EQ 0.
      COMMIT WORK.
      ev_message = 'OK'.
    ELSE.
      ev_message = 'NOK'.
    ENDIF.


  ENDMETHOD.
