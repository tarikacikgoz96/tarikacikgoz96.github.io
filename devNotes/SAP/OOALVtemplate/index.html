OO ALV Builder Template (Local Class)

Published: 14.06.2026 Updated: 22.06.2026

OO ALV için temel şablondur.

Bir SAP raporunda sadece bir ALV ekrana basılacak ise tercih edilebilir.
Veri çekme işleri dışarıda yapılır, (FORM, Metot, vb.) içine herhangi bir tipte itab gönderilir ve ekrana ALV basılmış olur.
Kullanımdan önce

  • Screen
  • GUI Status
  • GUI Title oluşturulmalı
  • LCL_ALV müşteri isteğine/rapora göre özelleştirilmelidir.

Şablon ve açıklamalar zamanla güncellenecektir.

Neden genellikle OO ALV tercih edilmelidir;

  • Simple ve Reuse ALV kullanılınca, ALV üzerinde tam yetkiniz olmaz.
  • Edite açık satır-hücre, satır-hücre boyama vb. geliştirmeleri diğer ALV çeşitlerinde yapamazsınız.
  • Müşteri her an yeni bir istekle gelebilir.

Temel çalıştırma aşağıdaki gibidir;

ZPROGRAM

REPORT ZOOALV_TEMPLATE.

INCLUDE zooalv_template_top.
include zooalv_template_cls.



START-OF-SELECTION.

  PERFORM GET_DATA.

  CREATE OBJECT GO_ALV.
  go_alv->set_data( REF #( gt_mara ) ).
  go_alv->run( ).

end-of-SELECTION.



FORM GET_DATA.
  SELECT *
    FROM MARA
    INTO TABLE GT_MARA
    WHERE MATNR IN S_MATNR.
ENDFORM.

LCL_ALV OO ALV Builder kodu;

  METHOD set_fcat.
    DATA: ls_fcat   LIKE LINE OF gt_FCAT,
          lv_colpos TYPE i.
    CLEAR : gt_fCAT.

    DATA: ls_tabdescr TYPE REF TO cl_abap_structdescr,
          lv_data     TYPE REF TO data,
          lt_dfies    TYPE ddfields,
          ls_dfies    TYPE dfies.

    FIELD-SYMBOLS: <gt_itab> TYPE STANDARD TABLE.
    ASSIGN gr_itab->* TO <gt_itab>.
    CREATE DATA lv_data LIKE LINE OF <gt_itab>.
    ls_tabdescr ?= cl_abap_structdescr=>describe_by_data_ref( lv_data ).
    lt_dfies =
     cl_salv_data_descr=>read_structdescr( r_structdescr = ls_tabdescr ).

    LOOP AT lt_dfies INTO ls_dfies.
      CLEAR ls_fcat.
      MOVE-CORRESPONDING ls_dfies TO ls_fcat.
      CASE ls_fcat-fieldname.
        WHEN 'MANDT'.
          ls_fcat-tech = 'X'.
        WHEN 'STATUSICON'.
          ls_fcat-icon      = 'X'.
          ls_fcat-hotspot   = 'X'.
          ls_fcat-reptext   = 'Status'(006).
          ls_fcat-scrtext_s = 'Status'(005).
          ls_fcat-scrtext_m = 'Status'(005).
          ls_fcat-scrtext_l = 'Status'(005).
       "......
        WHEN OTHERS.
          "ls_fcat-coltext = ls_fcat-fieldname.
      ENDCASE.
      APPEND ls_fcat TO gt_FCAT.
    ENDLOOP.

  ENDMETHOD.
