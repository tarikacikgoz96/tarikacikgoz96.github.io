SAP ABAP Development Workbench Tcodes

Published: 11.06.2026 Updated: 13.06.2026

SAP ABAP yazılımcılarının bilmesi gereken Tcode’lardır.
SAP ABAP dünyasında çok fazla tcode mevcuttur. Geliştirme yaparken unutmamak, tekrar hatırlamak adına aktif kullanılan tcodeların genel listesidir.
Liste zamanla güncellenecektir.

Development Tcodes

  • SE80 : Paket yönetim ekranı (Tüm dev içeriği)
  • SE38 : Program
  • SE24 : Class
  • SE37 : Function
  • SE11 : Table, Data Element, Domain, Structure, View (DB elemanları)
  • SE16, SE16N : Tablo içeriği verileri
  • SE14 : Tablo adjust etme, tablo içeriğini tamamen silme
  • SE41 : GUI-Status, Title ekranı
  • SE09 : Sistem request listesi / Req. oluşturma
  • STMS : Sistem request Kuyruğu (DEV, TEST, CANLI)
  • SM37 : Job Görüntüleme, inceleme
  • SM36 : Job oluşturma
  • SM12 : Kilit kayıtları - Edite Kitlenen programları/nesneleri kullanıcı bazında açma
  • SE03 : Transport Organizer - Hangi nesne hangi transportta/requestte
  • SE93 : Tcode ekranı
  • SE43 : Area Menu ekranı (Pakete özel menü hazırlama)
  • SMARTFORMS : Smartform oluşturma, değiştirme, oluşturma
  • SMARTSTYLES : Font, Yazı tipi vb. ayarlama
  • SE78 : Logo yükleme ekranı
  • SOST : Gönderilen maillerin kontrolü
  • STRUST : API Bağlantılar için sertifika yükleme
  • /IWFND/GW_CLIENT : OData, RestAPI test, oluşturma Ekranı
  • SEGW : OData Service listesi, ekranı
  • SM59 : HTTP HTTPS API RFC vb. bağlantı kontrol ekranı
  • SM50 : Aktif arkada çalışan programların listesi (O anki sistem/sunucu)
  • SE63 : Program metin çevirileri (Translate)
  • SE51 : Screen Painter ekranı
  • SM30 : Table Maintence Ekranı
  • /n : Üst ekrana geçiş, geri
  • /h : Anında Debug
  • AL11 : Sistem dosya yolları
  • CG3Y : Sistemden dosya indirme (Req. vb)
  • CG3Z : Sisteme dosya yükleme (Req vb.)

Birçok tcode/ekrana birçok yerden kod yazmadan da ulaşılabilmektedir.
